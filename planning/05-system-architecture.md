## 2. Authentication Flow

```
 Client                   API Server              Database
   │                          │                       │
   │  POST /api/auth/login     │                       │
   │  { email, password }      │                       │
   │─────────────────────────►│                       │
   │                          │  SELECT user WHERE    │
   │                          │  email = ?            │
   │                          │──────────────────────►│
   │                          │                       │
   │                          │◄──────────────────────│
   │                          │  user row             │
   │                          │                       │
   │                          │  bcrypt.compare()     │
   │                          │  jwt.sign(payload)    │
   │                          │                       │
   │◄─────────────────────────│                       │
   │  { user, token }         │                       │
   │                          │                       │
   │  (store token in         │                       │
   │   localStorage)          │                       │
   │                          │                       │
   │  GET /api/articles        │                       │
   │  Authorization: Bearer.. │                       │
   │─────────────────────────►│                       │
   │                          │  jwt.verify(token)    │
   │                          │  req.user = payload   │
   │                          │──────────────────────►│
   │                          │                       │
   │◄─────────────────────────│                       │
   │  { items, total }        │                       │
```

## 5. Monorepo Package Structure

```
workspace root (pnpm workspaces)
│
├── lib/                       ← Shared libraries (composite TypeScript, emit declarations)
│   ├── db/                    — Drizzle ORM schema, pool, migrations
│   ├── api-spec/              — OpenAPI YAML + Orval codegen config
│   ├── api-zod/               — Generated Zod validation schemas
│   └── api-client-react/      — Generated React Query hooks + custom-fetch
│
├── artifacts/                 ← Deployable applications (leaf packages, no emit)
│   ├── api-server/            — Express 5 REST API (port 8080, path /api)
│   └── news-portal/           — React + Vite SPA (port 21440, path /)
│
└── scripts/                   ← Utility scripts (seed data, migrations)
```
